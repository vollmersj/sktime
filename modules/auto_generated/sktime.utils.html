sktime.utils¶

sktime.utils.all_estimators(estimator_type=None)[source][source]¶

Get a list of all estimators from sktime.

This function crawls the module and gets all classes that inherit from sktime’s and sklearn’s base classes.

Not included are: the base classes themselves, classes defined in test modules.

Parameters

estimator_type (string, list of string, optional (default=None)) – Which kind of estimators should be returned. - If None, no filter is applied and all estimators are returned. - Possible values are ‘classifier’, ‘regressor’, ‘transformer’ and ‘forecaster’ to get estimators only of these specific types, or a list of these to get the estimators that fit at least one of the types.

Returns

estimators – List of (name, class), where name is the class name as string and class is the actual class.

Return type

list of tuples

References

..[1] Modified version from scikit-learn’s all_estimators() in sklearn.utils.__init__.py
