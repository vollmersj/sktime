exception sktime.exceptions.FitFailedWarning[source][source]¶

Bases: RuntimeWarning

Warning class used if there is an error while fitting the estimator. This Warning is used in meta estimators GridSearchCV and RandomizedSearchCV and the cross-validation helper function cross_val_score to warn when there is an error while fitting the estimator.

exception sktime.exceptions.NotEvaluatedError[source][source]¶

Bases: ValueError, AttributeError

exception sktime.exceptions.NotFittedError[source][source]¶

Bases: ValueError, AttributeError

Exception class to raise if estimator is used before fitting. This class inherits from both ValueError and AttributeError to help with exception handling and backward compatibility.
